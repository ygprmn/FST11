GUI - Graphical User Interface: Icons/Buttons/Menus
CLI - Command-line Interface: Text-based, interacting and managing of files/folders

- Make directory
mkdir "directory name"

- Remove directory
rmdir "directory name"

- Print/Present working directory
pwd

- Change directory
cd "directory path"

- Go back one directory
cd ..

- Create file
touch "file name"

- Remove file
rm "file name"

- Copy files
cp "source file" "destination path" 

- Move files
mv "source file" "destination path

- Rename files
mv "old file name" "new file name"
